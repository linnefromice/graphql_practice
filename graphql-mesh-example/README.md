# graphql-mesh-example

## Command

### Environmental construction

- yarn init -y
- yarn add graphql
- yarn add @graphql-mesh/runtime
- yarn add @graphql-mesh/cli
- yarn add @graphql-mesh/openapi
- yarn graphql-mesh serve
